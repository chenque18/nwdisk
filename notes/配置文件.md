
# 说明
将服务器ip和port(端口)写入client.config文件中
将配置文件作为资源文件添加到qt项目中

在项目目录下新建client.config文件
127.0.0.1
8888

# 思路
先在头文件定义ip和port成员变量，使用QFile类读取文件，打开方式为只读，读取全部，转化为字符串，去掉\r\n，空格分割，得到ip和port，分别存放到成员变量中

# 新建qt项目
选择Qt Widgets Application
设置项目名称TcpClient，基类选择QWidget，类名改为TcpClient，添加qt resource文件，命名为client.qrc，将client.config添加到资源文件中

tcpclient.h
```cpp
//添加头文件
#include<QFile>
//在class TcpClient : public QWidget添加
public:
    //初始化加载配置文件
    void loadConfig();

private:
    //存放ip和port
    QString m_strIP;
    quint16 m_usPort;
```

tcpclient.cpp
```cpp
#include <QByteArray>
#include <QDebug>
#include <QMessageBox>
//定义loadConfig函数
void TcpClient::loadConfig()
{
    QFile file(":/client.config");
    //设置只读打开
    if(file.open(QIODevice::ReadOnly)){
        //全部读取,返回字节，添加头文件
        QByteArray data=file.readAll();
        //转化为字符串
        QString stdata=data.toStdString().c_str();
        //qDebug()<<stdata;
        //如果打开成功，显示"127.0.0.1\r\n8888"
        file.close();
        //去掉\r\n
        stdata.replace("/r/n","");
        //qDebug()<<stdata;
        //通过空格分割,得到ip和port
        QStringList stlist=stdata.split(" ");
# if0
        for(auto i=0;i<stlist.size();i++){
            qDebug()<<stlist[i];
        }
# endif
        //将ip和port分别存放到成员变量中
        m_strIP=stlist[0];
        m_usPort=stlist[1].toUShort();
        

    }
    else{
        //错误提示
        QMessageBox::critical(this,"错误","配置文件打开失败");
    }
}
```
