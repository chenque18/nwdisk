- 使用单例模式  
opewidget.h/cpp
```cpp
public:
    static OpeWidget& getInstance();

OpeWidget &OpeWidget::getInstance()
{
    static OpeWidget instance;
    return instance;
}
```
在成功登录后，显示好友操作界面
