
# 用户表
| 字段 | 类型         | 约束条件  | 其他       |
| ---- | ------------| -------- | ---------|
| id   | 整型（int）  | 主键      | 自动增长   |
| name | varchar(32) | not null |          |
| pwd  | varchar(32) | not null |          |

# 好友表
| 字段     | 类型        | 约束条件 | 其他 |
| ---------| -----------| --------| --- |
| id       | 整型（int） | 主键     | 外键 |
| friendId | 整型（int） | 主键     | 外键 |

```sql
create database network_disk;
use network_disk;
create table user_info(
    id int primary key auto_increment,
    name varchar(32) not null,
    pwd varchar(32) not null
);
create table friend_info(
    id int,
    friendid int,
    primary key (id, friendid),
    foreign key (id) references user_info(id),
    foreign key (friendid) references user_info(id)
);
```
